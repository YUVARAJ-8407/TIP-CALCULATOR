# TIP-CALCULATOR

\# Tip Calculator



A simple Python program to calculate the total bill including tip

and split it among multiple people. 



\## Features

\- Handles invalid inputs using try-except

\- Allows continuous input until valid numbers are provided

\- Splits the bill per person



\## How to Run

1\. Open the terminal or your Python IDE

2\. Run `tip-calculator.py`

3\. Follow the prompts to enter bill amount, tip percentage, and number of people

4\. Get the total bill and amount per person
